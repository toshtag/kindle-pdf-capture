# kindle-pdf-capture

Capture Kindle for Mac pages via screen recording and build a high-quality PDF (with optional OCR).

> 📚 Languages: [English](docs/en/) · [日本語](docs/ja/)

## Quick start

```bash
uv sync
uv run kpc --help
```
