# Legal Notice

## How this tool works

kindle-pdf-capture captures Kindle for Mac pages using the macOS Screen
Recording API (CGWindowListCreateImage). It takes screenshots of the visible
window content and assembles them into a PDF.

**This tool does not:**
- Decrypt or extract DRM-protected content from Kindle files
- Bypass any technical protection measure
- Access Kindle's internal file formats or databases
- Transmit any data externally

## Personal use

This tool is intended for personal archiving purposes — for example, to create
a backup of a book you have legitimately purchased, for offline reading on a
