# 法的事項

## 本ツールの動作原理

kindle-pdf-capture は macOS の画面収録 API（CGWindowListCreateImage）を使って Kindle for Mac のウィンドウをキャプチャし、PDF に組み立てます。

**本ツールは以下を一切行いません：**

- Kindle ファイルの DRM 復号または内部データの抽出
- 技術的保護手段の回避
- Kindle の内部ファイル形式やデータベースへのアクセス
- データの外部送信
