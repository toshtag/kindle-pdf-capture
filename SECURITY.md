# Security Policy

## Reporting a vulnerability

Please do not report security vulnerabilities via public GitHub issues.

Instead, open a [GitHub Security Advisory](https://github.com/toshtag/kindle-pdf-capture/security/advisories/new) or email the maintainer directly.

Include as much detail as possible: steps to reproduce, potential impact, and any suggested fixes.

We will acknowledge receipt within 48 hours and aim to release a fix within 14 days for confirmed vulnerabilities.
